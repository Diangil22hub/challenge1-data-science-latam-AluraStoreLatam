# challenge1-data-science-latam-AluraStoreLatam

Este proyecto contiene un análisis detallado del desempeño de cuatro tiendas comerciales, evaluando métricas clave como ingresos, ventas por categoría, calificaciones de clientes, costos de envío y volumen de ventas. El análisis fue realizado en Google Colab utilizando Python para procesar y visualizar los datos.

# Descripción del Proyecto

El objetivo de este análisis es evaluar el rendimiento de cuatro tiendas para determinar cuáles son las más rentables y sostenibles a futuro, y tomar decisiones estratégicas sobre posibles cierres o reestructuraciones.

# Dataset
La información analizada incluye:

- Ingresos totales por tienda
- Ventas por categoría (Electrónicos, Electrodomésticos, Muebles, Instrumentos musicales, Deportes, Juguetes, Hogar, Libros)
- Calificación promedio por tienda
- Costo de envío promedio por tienda
- Cantidad vendida por categoría

# Conclusiones Clave

- Tienda 1 es la más sólida: lidera en ingresos, tiene la mejor calificación y domina categorías de alto valor como electrónicos y electrodomésticos.
- Tienda 4 presenta una paradoja: lidera en volumen de ventas y tiene el costo de envío más bajo, pero sus ingresos son los más bajos y su calificación es alarmantemente pobre (2.5).
- Tienda 2 y 3 tienen un desempeño moderado, sin grandes debilidades pero sin destacar en ninguna métrica clave.

# Tecnologías Utilizadas
- Python 3.x
- Google Colab
- Pandas – limpieza y análisis de datos
- Matplotlib / Seaborn – visualización de datos
- NumPy – cálculos numéricos
